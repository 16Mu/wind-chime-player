## 🎯 核心功能

### 主要特性
- ☁️ **远程音乐访问** - 无需下载，直接播放云端音乐
- 🔄 **智能同步** - 自动扫描和更新音乐库
- 📁 **文件浏览** - 方便的云端文件浏览器
- 🎵 **流式播放** - 边下边播，流畅体验
- 💾 **智能缓存** - 自动管理缓存，优化性能

### 支持的服务
- ✅ Nextcloud
- ✅ ownCloud
- ✅ Seafile (需WebDAV插件)
- ✅ 其他标准WebDAV服务
